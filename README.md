# EsempioThymeleafSpringMVC
Il progetto EsempioThymeleafSpringMVC contiene alcuni esempi di utilizzo di Thymeleaf con Spring MVC.<br>
Di seguito una lista completa degli esempi presenti in tale progetto:<br>
1. Visualizzazione in una pagina jsp di una lista di oggetti restituita da un controller Spring MVC (utilizzo dei costrutti th:each e th:text)
2. Chiamata ad un metodo del controller che aggiunge un libro alla lista iniziale (utilizzo del costrutto th:href)
3. Form di creazione di un nuovo libro (utilizzo dei costrutti th:action e th:object)

## Configurazioni necessarie
Per provare il progetto è necessario installare Tomcat sul WorkSpace
